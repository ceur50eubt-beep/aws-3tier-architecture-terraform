# aws-3tier-architecture-terraform

AWSにおけるベーシックかつ堅牢なWeb3層（Web/AP/DB）アーキテクチャをTerraformで実装したベースラインテンプレートです。

## 🏛️ アーキテクチャ概要
- **VPC / ネットワーク分離**: イングレス層（Public）、アプリケーション層（Private）、データ層（Private）をサブネットレベルで適切に分離し、最小権限のネットワーク動線を確保。
- **検証用ベースラインとしての位置づけ**: あえてS3等のリモートバックエンドや複雑なモジュール構成を排除し、ローカル環境で即座に挙動検証ができる「引き算の設計」を採用しています。

## 🎯 本リポジトリの位置づけ
本リポジトリは独立した基礎スタックの証明であると同時に、別リポジトリで公開している**「AI-Infra-Orchestrator（LLMによる自律デバッグエンジン）」**がエラーを検知して IaC コードを動的修復する際の、シンプルかつ確実な「検証用ターゲット（ベースライン）」としての役割を兼ねています。
